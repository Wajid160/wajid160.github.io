    AI Weather Agent

    This project demonstrates an AI-powered agent capable of fetching and reporting real-time weather conditions and daily forecasts for locations worldwide. Users interact with the agent through a simple chat interface provided by Chainlit. When a user specifies a city, the agent utilizes a tool connected to a real-world weather API to retrieve the latest data and presents it back to the user.

    Live Demo / How to Use

    How to use: Interact with the Chainlit chat interface embedded above (or linked). Simply type the name of a city (e.g., "What's the weather like in London?") and press Enter. The agent will process your request, use its tool to fetch the weather data, and respond with the current conditions and forecast.

    Project Documentation

    Overview

    The AI Weather Agent is built using Python and leverages the OpenAI Agents SDK to create an intelligent agent capable of understanding user requests and utilizing external tools. The core functionality revolves around fetching weather data. A Chainlit front-end provides a user-friendly chat interface for interaction. The agent is designed to be extensible, potentially incorporating more tools or features in the future.

    Features

    • Natural Language Interaction: Users can ask for weather information in a conversational manner via the Chainlit interface.
    • Real-time Weather Data: Utilizes a tool connected to a live weather API to provide current conditions.
    • Daily Forecasts: Capable of retrieving and displaying daily weather forecasts for the requested location.
    • Global Coverage: Can fetch weather data for cities around the world, dependent on the underlying weather API's coverage.
